# MERN Project Management Tool

A full stack project management application built using the MERN stack.

## Features
- User authentication (JWT)
- Create, update, delete projects
- Task assignment & tracking
- Role-based access

## Tech Stack
- MongoDB Atlas
- Express.js
- React.js
- Node.js
- Redux Toolkit

## Setup Instructions

### Backend
```bash
cd Backend
npm install
npm start

### Frontend
cd Frontend
npm install
npm start
